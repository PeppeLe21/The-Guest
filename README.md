# 🕵️ The Guest

> **Investigative Adventure · Social Deduction · AI-Powered NPCs**

📄 [Game Design Document](GDD_TheGuest.pdf)

**The Guest** is a noir deduction game developed in **Unity 6** for PC, where every conversation with a suspect is generated in real time by a Large Language Model. Interrogate, infiltrate, deceive — the truth is hidden behind words.

---

## 📖 Story

The adventure is set in Borgoscuro, a mountain village shrouded in fog. Count Amedeo Valli, the most powerful man in town, has been found dead in his study. The official investigation seems stalled and the residents refuse to cooperate.

A police officer has called you in: you are an outside private investigator, with no ties to the community. Your task is to knock on doors, gather clues and unmask the killer before the truth is buried forever.

---

## 🎮 Gameplay

The Guest revolves around an investigative core loop:

```
Observe the suspect
      ↓
Choose the right disguise (Social Stealth)
      ↓
Interrogate with free dialogue (LLM)
      ↓
Earn trust → Gain access to their home
      ↓
Collect physical evidence (procedural spawn)
      ↓
Analyse data in the Investigation Notebook
      ↓
Accuse the culprit
```

### 🎭 Social Stealth (Disguise System)

Stealth in The Guest is not physical — it is **social**. Wearing the right disguise before knocking on a suspect's door radically changes their attitude.

- Each costume carries a different initial trust value for each NPC
- The active disguise is injected into the AI System Prompt, shaping the model's behaviour

### 💬 Dynamic LLM Dialogue

Unlike classic dialogue trees, in The Guest you can type **anything**. The LLM-based NLP system:

- Maintains a **persistent memory** of the conversation
- Performs **Sentiment Analysis** to evaluate the player's approach (empathetic, aggressive, manipulative)
- Dynamically updates the **Trust Score** (0–100) of the NPC
- Generates responses consistent with each character's personality and secrets

> The minimum trust threshold to gain access to a home is **65/100**. Reaching it does not automatically grant entry — you still have to convince the character with your words.

### 📓 Investigation Notebook

The notebook updates automatically after every significant conversation. Summaries are **not pre-written**: they are generated by a second LLM that analyses the conversation log and extracts key statements, alibis and contradictions.

### 🔍 Evidence Collection

Physical clues are found **only inside private homes**, never in the street. The position of each item is determined procedurally every run, ensuring replayability.

## 🤖 AI Architecture

The AI system is **modular and provider-agnostic**: the codebase is designed to interface with any LLM API without rewriting gameplay logic.

### Production Models (via Groq Cloud)

| Model | Task |
|---|---|
| `moonshotai/kimi-k2-instruct` | NPC personality simulation and real-time dialogue |
| `qwen/qwen3-32b` | Notebook summary generation |

### Conversation Pipeline

```
[Player Input]
      ↓
[LLMManager] assembles System Prompt:
  - NPC psychological profile (NPCProfile ScriptableObject)
  - Player's active disguise (Social Stealth injection)
  - Conversation history (NPCMemory)
      ↓
[Groq API] → kimi-k2-instruct
      ↓
[LLMResponseParser] splits the JSON response:
  - Visible text → NPC speech bubble (DialogueUI)
  - Metadata → Trust Score update, door unlock, quest advance
      ↓ (in background, without blocking gameplay)
[qwen3-32b] → summary → NotebookManager
```

## 🚀 Getting Started

> ⚠️ **Prerequisite:** API keys have been removed from the code for security reasons. You must enter them manually in the Unity Inspector before running the game.

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/the-guest.git
```

### 2. Open with Unity Hub

- Open **Unity Hub** → *Add project from disk*
- Select the `The Guest` folder
- Use Unity version **6000.0.58f2** (Unity 6)

> The first launch will import all packages from `Packages/manifest.json` automatically (this may take a few minutes).

### 3. ⚠️ Configure the API Key (required)

The game uses **Groq Cloud** as the primary AI provider.

1. Get a free key at [console.groq.com/keys](https://console.groq.com/keys)
2. Open the `Demo.unity` scene
3. In the hierarchy, find the **GameObject** with the `GroqClient` component
4. In the **Inspector**, paste your key (starts with `gsk_`) into the `Api Key` field
5. **Do not save the key in the code** — keep it in the Inspector only

### 4. Run the game

Open the `Menu_iniziale.unity` scene and press **Play**.

## 🎨 Art Style

The visual style is **Low Poly**, built with third-party assets selected for visual consistency:

- **Low Poly House Pack** (Islide) — Interactable houses
- **SimplePoly City** — Background city elements
- **Low Poly Isometric Rooms** (Sketchfab) — Interior environments
- **LowPoly Models** (Quaternius) — Player character, NPCs and animations
